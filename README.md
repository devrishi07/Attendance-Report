# Attendance-Report
Fetching required attendance data from the student portal by remote logging in using requests library, beautifulsoup to parse and find the attendence data required. 

Run the file and you will be greeted by this:

<img width="1077" height="95" alt="image" src="https://github.com/user-attachments/assets/070f1c52-46e1-4fe2-9b91-50050e84acfc" alt="terminal when the code is run"/>

A file named captcha.jpg is loaded in you folder, enter your credentials and captcha from the generated file without any errors.

If everything went well your attendance report will be displayed like this:
